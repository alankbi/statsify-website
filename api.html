Website Endpoint

http://website-visualizer.herokuapp.com/website

Options:

url (required)


The desired url.

depth (default = 1, 0 ≤ depth ≤ 8)


The maximum depth of links to recursively crawl through before stopping.

Example Call:

http://website-visualizer.herokuapp.com/website?url=website-visualizer.com&depth=1

Response:
